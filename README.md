# 🧠 Inferential Statistics Project

## 📌 Overview

This project demonstrates the application of **inferential statistics techniques** on a health dataset.
It includes hypothesis testing, confidence intervals, and statistical analysis using Python.

The project is divided into two parts:

* **Part A** → Theory
* **Part B** → Practical implementation using Python (Jupyter Notebook)

---

## 📊 Dataset

The dataset contains **1200 synthetic health records** with the following features:

* **Demographics**: Age, Gender, Region
* **Lifestyle Factors**: Smoking Status, Exercise Frequency
* **Health Metrics**: BMI, Blood Pressure, Cholesterol, Glucose
* **Disease Indicators**: Diabetes, Hypertension

📁 File: `health_dataset.csv`

## 📗 Part B – Practical Implementation

### 🔍 Tasks Performed

### 1. Data Loading & Exploration

* Loaded dataset using Pandas
* Viewed sample records

---

### 2. Hypothesis Formulation

* Defined Null Hypothesis (H0) and Alternative Hypothesis (H1)

---

### 3. Confidence Interval

* Calculated 95% confidence interval for blood pressure

---

### 4. T-Test

* Compared glucose levels between smokers and non-smokers
* Calculated:

  * T-statistic
  * P-value
  * Critical value
* Decision: Accept/Reject H0

---

### 5. Chi-Square Test

* Tested relationship between smoking and diabetes
* Calculated:

  * Chi-square statistic
  * P-value
  * Critical value

---

### 6. ANOVA Test

* Compared blood pressure across age groups
* Calculated:

  * F-statistic
  * P-value
  * Critical value

---

### 7. Correlation & Covariance

* Analyzed relationship between age and BMI

## ✅ Results & Interpretation

* Statistical tests were performed using a significance level of **α = 0.05**
* Most tests resulted in **acceptance of the null hypothesis**, indicating no statistically significant relationships in the dataset
* This outcome is expected due to the **synthetic nature of the data**
